Welcome! This folder contains everything you need to run the 
RakeBob Multiplayer game.

┌──────────────────────────────────────────────────────────────────┐
│ QUICK START (3 STEPS)                                            │
└──────────────────────────────────────────────────────────────────┘

1. Install Node.js from: https://nodejs.org/

2. Open terminal/command prompt in THIS folder and run:
   
   npm install
   npm start

3. Open browser and go to: http://localhost:3000

That's it! You're ready to play!

└──────────────────────────────────────────────────────────────────┘

LOCAL NETWORK:
1. Start server on your computer
2. Find your IP address (ipconfig on Windows)
3. Friends connect to: http://YOUR_IP:3000

ONLINE (Deploy to internet):
1. Push code to GitHub
2. Deploy on Railway.app or Render.com (free!)
3. Share the public URL with anyone worldwide
